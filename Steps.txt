Steps to run applications

1. Open BankPro and configure connection string as per local DB settings
2. Open Nuget package manager console
3. Select project BankPro.Data
4. Execute script 'Update-database'
5. Open DB insertion script with projects
6. Execute DB scripts
7. Run BankPro ( Provide header key 'Token' with value 'BankPro Validation Token')
